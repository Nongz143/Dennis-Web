# Dennis Felicia's Web Project – Module 9 Skills Demo

## Overview

This project demonstrates my ability to build a multi-page responsive website using HTML, CSS, and JavaScript. It includes semantic structure, grid and flexbox layout strategies, a Google Font, FontAwesome icons, a favicon, and a dynamic back-to-top button. The site is hosted on GitHub Pages:  
🔗 [Dennis Felicia's Website](https://nongz143.github.io/Dennis-Web/)

## Layout Strategy

I used **CSS Grid** to organize the main content cards and **Flexbox** within individual sections for alignment and spacing. The `.app-grid` container centers the layout and adapts to screen size using `auto-fit` and `minmax`.  
Responsive design is achieved with media queries to ensure readability on mobile devices.
